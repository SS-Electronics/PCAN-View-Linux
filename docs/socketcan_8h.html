Detailed Description

SocketCAN back-end interface (Linux PF_CAN / SOCK_RAW).

Declares the per-connection context (socketcan_ctx_t) and the low-level SocketCAN operations that implement the generic can_driver_t vtable. Interface bring-up/configuration is performed via the ip utility (elevated through pkexec when required), avoiding a bespoke Netlink implementation.
