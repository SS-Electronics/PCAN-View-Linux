Detailed Description

A single decoded-able signal within a message.

Bit positions follow the DBC convention: start_bit is the position of the signal's least-significant bit for Intel layout, or of its most-significant bit for Motorola layout, in byte-wise LSB0 numbering (bit 0 = LSB of byte 0, bit 7 = MSB of byte 0, bit 8 = LSB of byte 1, …).
