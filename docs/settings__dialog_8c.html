Detailed Description

Modal CAN connection-settings dialog.

Presents the connection options and, on confirmation, writes them into g_app and triggers app_do_connect. The user can choose:

  • the CAN interface (auto-discovered from /sys/class/net, with vcan0 always offered as a fallback),
  • the nominal bit rate,
  • CAN FD mode and its data bit rate,
  • listen-only mode.
