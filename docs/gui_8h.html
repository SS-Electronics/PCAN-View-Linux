Detailed Description

GTK GUI widget registry and the application's GUI/thread API.

Declares the trace-view column layout, the global widget bundle (gui_widgets_t / g_gui), and every public function implemented by the gui/ sources: window construction, trace/statistics updates, dialogs, value formatting, the RX/TX/stats thread entry points, and the connect/disconnect lifecycle helpers.

Column indices of the trace GtkListStore.

Columns up to TCOL_COUNT are visible; the remainder are hidden helper columns used for sorting, row roll-up, and re-formatting.
