Detailed Description

CAN trace GtkTreeView, value formatting, and statistics rendering.

Builds and maintains the receive/transmit trace list and its backing GtkListStore, including row roll-up (deduplication) by CAN ID, interval and frequency computation, and re-formatting when the ID/data display mode changes. Also provides the shared value-formatting helpers (gui_format_id, gui_format_data, gui_msg_type_str, …), the statistics refresh (gui_update_stats), the status-bar helper, and the modal error dialog.
