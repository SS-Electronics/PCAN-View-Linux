Detailed Description

Generic CAN driver abstraction and the SocketCAN binding.

Implements the hardware-independent drv_can_* API declared in drv_can.h. Each call dispatches through the currently selected can_driver_t vtable (set by drv_can_init). This file also defines the built-in SocketCAN vtable, whose members are thin adapters that forward to the socketcan_* functions while binding the shared s_sck_ctx.
