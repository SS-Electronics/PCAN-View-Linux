Detailed Description

DBC (CAN database) parser and signal-bit decoder.

Implements dbc.h. The parser is a single-pass, line-oriented reader for the BO_ / SG_ records of the Vector DBC grammar. Anything it does not understand (value tables, attributes, comments, node lists, …) is ignored, so it loads real-world databases without choking while extracting exactly the structure the Signal Analysis view needs.

The decoder handles both bit layouts:

  • Intel (little-endian): bits are consumed LSB-first, ascending.
  • Motorola (big-endian): bits are consumed MSB-first using the classic "sawtooth" traversal across the byte-wise LSB0 numbering.
