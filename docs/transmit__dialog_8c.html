Detailed Description

Advanced ("Transmit...") CAN/CAN FD message composer window.

A non-modal secondary window for hand-crafting a single frame with full control over: identifier (hex) and standard/extended flag, data vs. remote (RTR) frame, CAN FD + BRS, DLC (0..8 classic, 0..64 FD), individual data bytes, and one-shot or cyclic transmission at a configurable interval. Frames are validated by build_frame and pushed onto g_app.tx_queue.
